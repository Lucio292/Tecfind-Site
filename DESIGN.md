## 🚀 Como Usar o Novo Design

### Para Prestadores:
1. Clique em **"👨‍🔧 Cadastrar como Prestador"**
2. Digite o seu serviço (ex: "Encanador experiente")
3. Adicione sua descrição e WhatsApp (opcional)
4. Defina sua localização no mapa
5. **Pronto!** Sua cor é aplicada automaticamente!

### Para Clientes:
1. Clique em **"👨‍💼 Cadastrar como Cliente"**
2. Digite o serviço que procura (ex: "Conserto de encanamento")
3. Digite seu bairro
4. **Sistema aplica cor automaticamente baseado na palavra-chave!**

## 💡 Como Adicionar Novos Serviços

Veja o arquivo **CORES.md** para instruções completas, ou:

### Rápido (2 minutos):

1. Abra **service-colors.css**
2. Adicione uma nova classe:
```css
.service-seu-servico {
    border-left: 5px solid #SEUCOLOR !important;
    background: linear-gradient(135deg, rgba(R, G, B, 0.05) 0%, rgba(R, G, B, 0.02) 100%) !important;
}
```

3. Abra **script.js**, procure `serviceKeywords` e adicione:
```javascript
'sua-palavra-chave': 'service-seu-servico'
```

**Pronto! Seu novo serviço tem cor!**
